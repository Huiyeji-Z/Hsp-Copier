# 剪贴板小组件 — 需求说明

## 一、项目概述

一个 Windows 桌面剪贴板历史小组件，名称为: Hsp Copier, 主体是一个可拖动、可调整尺寸的置顶小窗口。当窗口位于屏幕边缘且鼠标离开时，会以炫酷变形动画收缩为小悬浮球；鼠标移到悬浮球上时再以动画展开为窗口主体。

**技术栈**：希望有一个安装包exe, 用户执行后开始安装;  希望应用入口是一个可执行文件, 不依赖其它东西
**目标平台**：Windows 10 / 11
**数据存储**：用户目录（config.json + history.json + images/）

---

## 二、原始需求

### 1. 窗口与交互
- 小窗口，可随意修改尺寸、可随意拖动
- 圆角边框，永久置顶
- 位于屏幕边缘 + 鼠标离开 → 变形为小悬浮球（带炫酷变形动画）
- 鼠标移到悬浮球 → 变形回窗口主体（带动画）
- 固定按钮/取消固定，固定后强制不变形

### 2. 剪贴板功能
- 展示最近 N 次复制内容（默认 10，可配置）
- 支持文本和各类文件
- 长文本只展示前面部分，悬停 1s 后预览全部
- 文件只展示文件名
- 点击记录复制该内容/文件
- 点击后记录重排（可配置是否重排）

### 3. 个性化
- 默认毛玻璃背景，可改透明度
- 自定义窗口背景图（内置几张美观的 + 本地选择）
- 自定义悬浮球背景图/动图（内置图片 + GIF，如卡通人脸眨眼；本地选择）
- 4 种变形动画可选（缩放渐变 / 液体吸附 / 3D 翻转折叠 / 粒子消散重组）

### 4. 系统级功能
- 可以检查更新以及在线更新, 用户选择更新的话,从github仓库拉取最新文件, 更新途中软件不退出, 完成更新后新功能可直接使用, 这个方案是否可行?
- 系统托盘图标
- 记录复制来源应用
- 数据存到用户目录
